
# REST CRUD App (Student) — practice

A Spring Boot REST API that performs CRUD operations for a Student entity using MySQL and Spring Data JPA.

## Tech Stack
- Java
- Spring Boot (Web, Data JPA)
- MySQL
- Maven

## Student Model
Fields:
- id 
- name
- fees
- subName

## Database Setup (MySQL)
1. Start MySQL
2. CREATE DATABASE studentDb;
